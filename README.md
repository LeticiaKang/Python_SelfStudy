      - 개발자를 위한 파이썬
      - 처음 시작하는 파이썬
### 2. 학습 내용
      - 문법공부_파이썬 연습문제 풀기
      - 강의_파이썬으로 시작하는 데이터 사이언스

         - Flask_API를 이용해 웹으로 데이터 시각화하기
